# 🎲 幸运大转盘 - 抽奖运营页面

一个完整的前端抽奖运营页面，采用原生HTML、CSS、JavaScript开发，具有现代化的响应式设计和丰富的交互效果。

## ✨ 主要功能

### 🎯 转盘抽奖
- **精美转盘设计**：6个扇形区域，每个区域对应不同奖品
- **流畅旋转动画**：使用CSS3动画实现转盘旋转效果
- **概率控制系统**：可配置的奖品概率分配
- **视觉效果丰富**：包含指针、渐变背景、阴影等效果

### 📝 用户管理
- **信息收集**：用户姓名和手机号码验证
- **实时验证**：输入框实时验证，提供视觉反馈
- **数据持久化**：使用localStorage保存用户信息
- **自动加载**：页面刷新后自动恢复用户信息

### 🎮 游戏机制
- **每日限制**：每个用户每天最多3次抽奖机会
- **自动重置**：每日零点自动重置抽奖次数
- **抽奖历史**：记录用户所有抽奖记录
- **防作弊**：多重验证确保公平性

### 🎊 中奖体验
- **炫酷弹窗**：中奖结果弹窗展示
- **特效动画**：烟花、彩带等庆祝动画
- **音效支持**：预留音效接口（可选开启）
- **状态反馈**：实时显示按钮状态和剩余次数

## 💡 使用说明

### 用户操作流程
1. **填写信息**：在右侧输入真实姓名和手机号码
2. **确认信息**：点击"确认信息"按钮验证并保存
3. **开始抽奖**：点击"开始抽奖"按钮启动转盘
4. **查看结果**：等待转盘停止，查看中奖结果
5. **继续抽奖**：如有剩余次数，可继续参与

### 快捷操作
- **回车键**：在非输入框状态下，按回车键快速开始抽奖
- **ESC键**：关闭结果弹窗
- **自动保存**：用户信息和抽奖进度自动保存

## ⚙️ 配置选项

在 `script.js` 中的 `LOTTERY_CONFIG` 对象可以调整以下参数：

```javascript
const LOTTERY_CONFIG = {
    maxDailyAttempts: 3,    // 每日最大抽奖次数
    spinDuration: 4000,     // 转盘旋转时间（毫秒）
    minRotations: 5,        // 最少旋转圈数
    prizes: [...]           // 奖品配置
};
```

### 奖品概率调整
在 `prizes` 数组中修改每个奖品的 `probability` 值：
```javascript
{
    id: 1,
    name: 'iPhone 15',
    probability: 0.01,  // 1% = 0.01
    // ...其他配置
}
```
